How to View the YouTube Videos for the Top 100 Songs of the Year

This posts may help you to watch the music videos for the Top 100 songs of 2008 (I think they're the second half of the list, below) without having to look them up or open them individually. Here are the steps, but with luck you can skip most of these and go to the list of URLs, below. 1. Prepare a list of URLs. Do this using e.g., the list at About.com . In your browser, view the page source. Copy it all as text to e.g., Microsoft Word. Sort the paragraphs; delete everything that doesn't have a reference to YouTube; do some global searching and replacing (optionally using a macro) to remove everything except the YouTube URLs. 2. Put the URLs in batch file form. This means (a) do a global search and replace to put the appropriate command at the start of each line, with optional (probably advisable) quotation marks; (b) save in Notepad as a text file; (c) unless your browser can handle 200 video tabs at once, cut and paste this text file in pieces to another text file with a .BAT extension. Example: URL-List.txt has all 200 URLs, stated in command-line form (see 200 examples below); Open-URLs.bat has 20 of those 200; double-click on the latter and run it. 3. Get your browser in shape to run the batch file. I use Tab Mix Plus and set it to make sure that all tabs open in the same window, so I'm not opening dozens of distinct Firefox sessions, which would be a load on the computer. 4. Run the batch file. Wait for the videos to all download, and then you can watch them at your leisure. Repeat with the next set until you've worked through the whole list. I created a single list for 2007 and 2008 before I thought of writing this post, so the following list combines them both. About.com doesn't seem to have links to videos for anything before 2007 at this time, so that's why I did only those two years.
